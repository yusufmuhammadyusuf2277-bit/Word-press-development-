What is a WordPress Theme?

The theme controls your website’s appearance and layout. Themes can be free or premium.

Installing a Theme

  • Go to Appearance → Themes → Add New
  • Search or upload a theme
  • Click Install → Activate

Customizing Themes

Use the Customize option to change colors, fonts, headers, and layout settings without coding.

Child Themes

Child themes let you safely modify a theme without losing changes after updates.
